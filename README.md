# Military Asset Management System

## 📌 Project Overview
This project is a Military Asset Management System used to manage assets like weapons, vehicles, and equipment across multiple bases.

## 🚀 Features
- Asset Management
- Purchase Tracking
- Asset Transfers Between Bases
- Role-Based Access Control (Admin, Commander, Logistics)
- Secure Authentication (JWT)
- Dashboard with real-time data

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## 📂 Project Structure
military-asset-management-system/
├── military-frontend/
├── military-backend/

## ⚙️ Installation

### Backend
cd military-backend
npm install
npm start

### Frontend
cd military-frontend
npm install
npm start

## 🔐 Roles
- Admin → Full access
- Commander → Transfer access
- Logistics → View only

## 🌐 API Endpoints
- /api/auth
- /api/assets
- /api/purchases
- /api/transfers
